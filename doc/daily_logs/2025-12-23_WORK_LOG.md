### 2. 페이지 디자인 개선 (UI/UX)
- **문제**: 로그인/회원가입 폼이 화면 좌측 상단에 치우쳐 있어 시각적으로 불안정함.
- **해결**: Flexbox 유틸리티(`min-h-screen`, `items-center`, `justify-center`)를 사용하여 폼이 화면 정중앙에 위치하도록 수정했습니다.

### 3. 인증 리다이렉트 로직 강화 (Auth Redirect)
- **문제**: 로컬 개발 환경 또는 특정 상황에서 회원가입 인증 메일 클릭 시 `localhost`로 잘못 리다이렉트되는 현상 발생.
- **해결**: `src/app/(auth)/actions.ts`에서 리다이렉트 URL 결정 로직을 개선했습니다.
    - 우선순위: `NEXT_PUBLIC_SITE_URL` -> `VERCEL_URL` -> `origin` 헤더 -> `localhost`
    - 이를 통해 배포 환경과 로컬 환경 모두에서 안정적인 리다이렉트를 지원합니다.
